# **Software**

This folder contains the code that is utilized in our project. In particular, the code that controls our microcontroller (ESP32) and detects button pushes is featured in this folder. This code is important to accurately control the timing of the charging and discharging of our system.

All programming was completed using the [ESP-IDF](https://idf.espressif.com/), which is a development framework created specifically for the ESP32. Although the Arduino IDE or other similar methods can be used for the ESP32, the ESP-IDF exposes more low-level control over the microcontrollers, which was necessary for our project.
